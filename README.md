# Pashiz
A decentralized, peer-to-peer digital currency (actually a shitcoin) just for educational purposes.

## What does it mean?
**Pashiz** (Persian: پشیز) was the smallest currency in the Sassanid period.
Also it's a slang for anything cheap.
